# 🇮🇳 JAN GAN MAN

A futuristic rural India focused super-app built with Next.js 14.

---

# 🚀 Vision

JAN GAN MAN is designed to empower villages and small towns with modern digital services in one platform.

---

# ✨ Modules

- 🧑‍💼 Naukri
- 🛒 Dukaan
- 🏛 Sarkari Seva
- 🩺 Swasthya
- 📚 Shiksha
- 🌾 Krishi
- 🛡 Suraksha
- 🤖 AI Sahayak

---

# 🛠 Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP
- Three.js
- Lucide React
- Firebase
- Cloudinary
